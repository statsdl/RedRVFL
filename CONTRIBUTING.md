# Contributing to RedRVFL

Use focused pull requests and include tests for behavior changes.

## Development

```bash
pip install -e ".[dev]"
pytest
```

## Style

Code should follow PEP 8 and be formatted with Black-compatible settings from `pyproject.toml`.

## Issues

When reporting bugs, include:

- Python version
- RedRVFL version or commit
- Minimal code to reproduce the issue
- Expected and actual behavior
