RedRVFL
=======

RedRVFL provides RVFL, EDRVFL, and REDRVFL regressors for financial
time-series forecasting with Hyperopt/TPE tuning.

Installation
------------

.. code-block:: bash

   pip install redrvfl

Main Objects
------------

.. code-block:: python

   from redrvfl import RVFLRegressor, EDRVFLRegressor, REDRVFLRegressor

See the repository README for the DJI Yahoo Finance experiment.
